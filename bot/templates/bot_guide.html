Commands:

/help: bot will post a link to a list of its commands.

/new [/command] [response]: creates new command with response.

/edit [/command] [response]: edits existing command with new response.

/delete [/command]: deletes existing command.

/newrand [command] [response]: creates a new random command whose responses will be selected from randomly (more than one response may exist per each random command).

/random [command]: bot responds with randomly chosen response from random commands with this name.
